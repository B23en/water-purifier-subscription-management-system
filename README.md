LG와 국립창원대학교가 연계하여 진행하는 에이전트 시스템 개발 프로젝트

## 📁 폴더 구조

| 폴더 | 설명 |
|------|------|
| `models/` | 비교분석 · 예측 모델 관련 파일 |
| `assets/` | 흐름도, 구조도 등 에셋 |
| `archived/` | 별도로 보관하는 파일 |
| `docs/` | 기획 · 명세 등 문서 |
